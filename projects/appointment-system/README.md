# Appointment System

## Problem

Small businesses often manage appointments manually (WhatsApp, phone calls, spreadsheets), causing missed bookings, double-booking, and wasted time.

## Solution

A lightweight appointment management system with roles, scheduling, and a simple admin view.

## Key Features

- Authentication + roles (Owner / Staff)
- Appointment scheduling (create, reschedule, cancel)
- Customer list + notes
- Basic daily/weekly view

## Tech Stack (planned)

- Backend: NestJS (TypeScript)
- Frontend: React + TypeScript
- Database: PostgreSQL
- Deployment: Docker (later)
